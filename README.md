# Newsletter-Sign-Up-Website

Mock Web app which uses the Mailchimp API to subscribe user emails to a newsletter.

# Languages and Tools:
**Front-End**
- JavaScript
- HTML
- CSS

**Back-End**
- Node.js
- Mailchimp API -> https://mailchimp.com/developer/
